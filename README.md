# Arduino-Tx-Rx


Arduino-Based One-Way Wireless Text Messaging System that utilizes RF (Radio Frequency) modules for communication between two Arduino devices: a transmitter and a receiver. 

The transmitter is responsible for sending messages encoded in numerical form, where each number corresponds to a specific letter of the alphabet based on a predefined scheme. 

Upon receiving the transmitted data, the receiver decodes the numerical values and converts them back into readable words or sentences. 

This system demonstrates basic wireless communication and data encoding and decoding using Arduino technology.
